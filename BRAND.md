# SciFlow Pro Brand System

This file records the public homepage brand direction so future pages can stay visually consistent.

## Logo

- Primary mark: `assets/brand/sciflow-mark.svg`
- Wordmark: `assets/brand/sciflow-wordmark.svg`
- Favicon: `favicon.svg`

The mark represents a scientific workflow chain: an S-shaped flow path, connected evidence nodes, and a structured data axis.

## Color Tokens

- Ink: `#0B1220`
- Surface: `#FFFFFF`
- Paper: `#F4F7FB`
- Primary blue: `#2F65D9`
- Cyan evidence link: `#22D3C5`
- Signal green: `#9FE870`
- Steel: `#18304F`

Use the palette as a scientific workbench system: dark ink for shell and hero, blue for primary actions, cyan for evidence links, and green only for confirmation or healthy state.

## Icon Style

- Stroke icons only.
- Rounded line caps and joins.
- Keep icons inside 36px square badges on cards.
- Use blue badges for project flow, cyan badges for local/evidence, green badges for validated status, and steel badges for documents or administration.

## Product Tour

The screenshot section should use continuous product-tour rows rather than a loose gallery. Each row pairs one real screenshot with one workflow explanation.
