# SciFlow Pro Downloads

This repository is only for public installer downloads of SciFlow Pro.

The application source code is private and is not published here.

## Downloads

Use the files attached to the latest GitHub Release:

- Windows: `SciFlow-Pro-Setup-1.2.0.exe`
- macOS Apple Silicon: `SciFlow-Pro-1.2.0-arm64.dmg`
- Android: `SciFlow-Pro-1.2.0-android.apk`
